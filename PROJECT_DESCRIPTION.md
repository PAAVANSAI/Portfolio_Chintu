# 📝 Project Description

## Portfolio Website — Arempula Chintu

### Overview
This is a **personal portfolio website** designed for **Arempula Chintu**, a recent Computer Science graduate from Sri Chaitanya Technical Campus, Hyderabad. The website showcases academic background, technical projects, skills, and contact information in a visually stunning, modern interface.

### Objective
To create a professional online presence that highlights qualifications, projects, and technical expertise — making it easy for recruiters, collaborators, and peers to learn about Chintu's capabilities and get in touch.

### Technology Used
| Technology | Purpose |
|-----------|---------|
| **HTML5** | Page structure and semantic markup |
| **CSS3** | Styling, animations, responsive layout, glassmorphism effects |
| **JavaScript** | Interactive features, particle system, scroll animations, typewriter effect |

> **No external frameworks or libraries** — built entirely with vanilla HTML, CSS, and JS.

### Design Highlights
- 🌑 **Dark theme** with purple and cyan gradient accents
- 🪟 **Glassmorphism** cards with frosted-glass borders
- 🌌 **Interactive particle background** with mouse-reactive connections
- 🎭 **Scroll-triggered reveal animations** for all sections
- 📊 **Animated skill bars** that fill on viewport entry
- 🃏 **3D tilt effect** on project cards
- 📱 **Fully responsive** — mobile hamburger menu, stacked layouts

### How to Run
