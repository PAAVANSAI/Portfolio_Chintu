# 🚀 Arempula Chintu — Portfolio Website

A modern, professional portfolio website built with pure **HTML**, **CSS**, and **JavaScript**. Features a premium dark glassmorphism design with smooth animations and interactive effects.

## ✨ Features

### 🎨 Design
- **Dark glassmorphism theme** with purple & cyan gradient accents
- **Premium typography** using Google Fonts (Space Grotesk + Inter)
- **Fully responsive** — works seamlessly on desktop, tablet, and mobile
- **Semantic HTML5** with SEO best practices

### 🎬 Animations & Interactions
| Effect | Description |
|--------|-------------|
| 🌌 Particle System | Interactive floating particles with connecting lines |
| ⌨️ Typewriter Effect | Cycles through roles — Software Developer, Java Enthusiast, etc. |
| 📜 Scroll Reveal | Elements fade and slide in as you scroll down |
| 📊 Skill Bar Fills | Animated progress bars for technical skills |
| 🔢 Counter Animation | Numbers count up with easing on the hero section |
| 🃏 3D Tilt | Project cards tilt toward cursor on hover |
| 🧲 Magnetic Buttons | Buttons subtly follow your cursor on hover |
| 💡 Cursor Glow | Ambient glow that follows the mouse |
| 🔗 Smooth Scrolling | Seamless navigation between sections |

## 📁 Project Structure

```
├── index.html      # Main HTML file with all sections
├── style.css       # Complete styling with animations & responsive design
├── script.js       # Interactions, particle system & scroll effects
└── README.md       # Project documentation
```

## 📋 Sections

1. **Hero** — Name, animated typewriter roles, key stats, and call-to-action buttons
2. **About** — Bio, personal info grid, avatar with graduation badge
3. **Education** — Animated vertical timeline (B.Tech → Intermediate → SSC)
4. **Projects** — Interactive cards for IoT Security and Crypto AI Analysis
5. **Skills** — Technical skill bars, framework chips, tools grid, and soft skills
6. **Contact** — Contact info cards + animated message form
7. **Footer** — Social links and copyright

---

## 🛠️ Tech Stack

- **HTML5** — Semantic markup and structure
- **CSS3** — Custom properties, glassmorphism, gradients, keyframe animations, flexbox & grid
- **Vanilla JavaScript** — Canvas API, Intersection Observer, DOM manipulation, requestAnimationFrame

> No frameworks, libraries, or build tools required. Just open `index.html` in a browser!

## 🚀 Getting Started

### Option 1: Direct Open
Simply double-click `index.html` or open it in any modern browser.

### Option 2: Live Server (VS Code)
1. Install the **Live Server** extension in VS Code
2. Right-click `index.html` → **Open with Live Server**

### Option 3: Clone & Run
```bash
git clone https://github.com/your-username/your-repo-name.git
cd your-repo-name
# Open index.html in your browser
```
